## Labedin 

Atividade com objetivo de ser apresentada como um currículo, parecido com as informações postas no Linkedin, em que contém:

-  Minha descrição
-  Email e endereço
-  Minha experiência profissional
-  Formação acadêmica
-  Redes sociais


### Tecnologia Utilizada
- React

### Link Surge 
https://crazy-addition.surge.sh/
