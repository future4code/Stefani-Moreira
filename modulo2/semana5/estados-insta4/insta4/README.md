## Insta4

Atividade com objetivo de reproduzir as funcionalidades do Instagram.

Nele é possível:

- Curtir e descurtir post
- Comentar
- Salvar post
- Criar um post

### Tecnologia Utilizada:
- React
### O que não funciona:
Aba de compartilhar post, apenas o ícone foi posto.
