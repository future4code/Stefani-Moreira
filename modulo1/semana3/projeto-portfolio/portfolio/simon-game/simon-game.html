Genius

O Genius (Simon-Game) tem a ideia de ser um jogo de memória em que você deve memorizar a sequência de cores, mas também pode ser feita pela sequência de sons, isso porque cada cor tem seu som característico.

Toda ideia é bem divertida e é legal de jogar também.

As tecnologias usadas foram

  • HTML
  • CSS
  • JavaScript
