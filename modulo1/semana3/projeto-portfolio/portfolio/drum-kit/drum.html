Drum Kit

O Drum Kit é um jogo que simula sons de bateria. Cada imagem emite um som e você pode combinar isso pra tocar algo. Os sons podem ser acionadas pelo mouse ou pelas teclas que são correspondentes a imagem.

As tecnologias usadas foram

  • HTML
  • CSS
  • JavaScript
