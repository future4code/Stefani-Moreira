TinCat

O TinCat é uma ideia de conectar pais de gatinhos para formar pares entre seus filhos. Tem toda uma estrutura para conversar com um gato e deixar bem claro seu propósito.

Esse é o meu maior orgulho até agora. Foi feito como atividade de um dos cursos que fiz, me dediquei bastante para fazê-lo e saiu bem legal.

  • CSS
  • Bootstrap
