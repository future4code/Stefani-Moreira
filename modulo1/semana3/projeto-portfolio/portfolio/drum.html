Drum Kit

O Drum Kit é um jogo que simula sons de bateria. Cada imagem emite um som e você pode combinar isso pra tocar algo. As imagens podem ser acionadas pelo mouse ou pelas teclas que correspondentes a imagem.

As tecnologias usadas foram

  • HTML
  • CSS
  • JavaScript
