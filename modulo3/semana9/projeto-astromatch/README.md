# Astromatch

Atividade com objetivo de reproduzir algumas funcionalidades do Tinder.

### Nele é possível:

- Procurar perfis para match
- Dar match com os perfis achados ou recusá-los
- Acessar a área de matches e encontrar as suas combinações
- Deletar os seus matches e reiniciar a procura de perfis

### Tecnologia Utilizada:
- React
- Axios
- Styled components
- Sweet Alert 2
